2023-06-03 1 min read # markdown # Windows软件

1.下载

官网：https://typoraio.cn

补丁：https://www.fahai.org/jszt/14.html

打包下载：https://www.123pan.com/s/A7ncVv-94Qrv.html提取码:2234

2.破解

关闭Typora软件，将winmm.dll放置于Typora安装目录下即可。
